## System Overview

Engram is a multi-layer AI memory and intelligence platform designed to provide persistent memory capabilities for AI agents. The system combines vector search, semantic memory, OSINT crawling, and a unified dashboard interface.

### Core Design Principles

1. **Modularity**: Each subproject is independently deployable
2. **Scalability**: Horizontal scaling via container orchestration
3. **Security**: Defense in depth with Tailscale VPN isolation
4. **Resilience**: Health checks, automatic recovery, and graceful degradation
5. **Multi-tenancy**: Native support for tenant isolation

### Technology Stack

| Layer | Technology | Purpose |
|-------|------------|---------|
| Vector Database | Weaviate 1.27 | Semantic search and storage |
| Cache Layer | Redis 7 | Session cache, rate limiting |
| Backend API | FastAPI (Python 3.12) | REST API, WebSocket |
| MCP Server | TypeScript (Node 20) | Model Context Protocol |
| Frontend | Next.js 15 (React 19) | Web dashboard |
| Authentication | Clerk | OAuth, user management |
| Proxy | Nginx | SSL termination, load balancing |
| Networking | Tailscale | Mesh VPN |
| Containerization | Docker Compose | Service orchestration |
